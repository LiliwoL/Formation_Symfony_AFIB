Installation de FOSUserBundle

`composer update`

3. Activer le Bundle

4. Créer une entité User

// src/AppBundle/Entity/User.php

5. Configurer config.yml
 
6. Insérer les routes FosUser

7. Mettre à jour la base de données

8. Configurer security.yml
